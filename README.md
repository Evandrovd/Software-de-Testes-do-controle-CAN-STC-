# Software-de-Testes-do-controle-CAN-STC-
Codigo escrito e testado para funcionamento. A nao ser que exista algum bug nao encontrado, voce nao deve coseguir queimar o seu motor usando este codigo sem alteracoes.

Programa escrito inicialmente para testar a funcionalidade do controle do motor via can, que sera posteriormente traduzido a uma ferramenta universal de envio de comandos no barramento can, para testes.
